## Problema

3 - Escreve um método static que inicialize e devolva um enumerável genérico contendo n cópias de um valor passado como parâmetro e tipo especificado como argumento genérico.

## Soluções

### Solução 1

```cs
    //Método para retornar enumerável genérico com n copias do objeto 
    public static IEnumerable<T> ReturnGenericEnum<T>(int n, T valorpassado)
    {
        //retornar T "n" numero de vezes
        for (int i = 0; i < n; i++)
        {
            yield return valorpassado;
        }
    }
```
